Intro

ExtentReports (by Anshoo Arora) is a HTML reporting library for Selenium WebDriver for Java and creates easy to use, attractive reports. It shows test and step summary, test steps and status in a toggle view for quick analysis.

Features:

  • Very simple to use
  • Beautiful and responsive UI
  • Provides dashboard for the entire run
  • Organizes all used media (screen-captures, videos) for the run in a single view (2.x)
  • Supports using parallel runs via TestNG (2.x)

Versions

A major difference between the two (besides the UI) is that version 2.0 is thread-safe and you can integrate it with your parallel run environment if you use TestNG or other test-runners. If your environment does not require this, or if you are running a single test at a time only, then you can choose either.

Below is a quick breakdown of both versions:

Version 1.x

Even though version 1 does not support multi-threaded runs, you may want to use it because:

  • Stable, already being used by many users
  • More feature intensive
  • Choice of Standard and Masonry grids
  • Many more options for customization than 2.0

Samples: Standard Grid, Masonry Grid.

Version 2.x

ExtentReports 2.0 is the future direction of this library. The main advantages of using this over versions 1.x are:

  • Thread safe - easily configurable for parallel classes or methods via TestNG
  • Clean and beautiful UI
  • Various options for dashboards, views for all used media type
  • Responsive grid compatible with all devices
  • Much better performance
  • More active development
