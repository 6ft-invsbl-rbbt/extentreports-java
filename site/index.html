Intro

ExtentReports 2.0 (by Anshoo Arora) is a HTML reporting library for Selenium WebDriver for Java and creates easy to use, attractive reports. It shows test and step summary, test steps and status in a toggle view for quick analysis.

Features:

  • Very simple to use
  • Beautiful and responsive UI
  • Dashboard for the entire run
